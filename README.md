# 🚀 ToolKit Pro - High Precision Web Tools

**ToolKit Pro** is a lightweight, high-performance suite of web utilities built with a "Performance-First" mindset. Developed entirely on a mobile environment (**Termux**), this project proves that powerful engineering doesn't require a high-end PC—just a sharp mind and the right logic.

---

## 🧠 The Core: Big.js Engine
Standard JavaScript floating-point math often fails in precision (e.g., `0.1 + 0.2 !== 0.3`). 
ToolKit Pro solves this by using the **Big.js engine**, ensuring:
* **Unlimited Precision:** Perfect for financial and scientific calculations.
* **Accuracy:** No more rounding errors.
* **Lightweight:** Minimal overhead for mobile users.

## 🛠️ Features Included
- **Calculator:** Powered by the Big.js engine for absolute accuracy.
- **Chess Game:** Built with pure Vanilla JS, featuring AI and 2P modes.
- **Currency Converter:** Supporting 160+ real-time global currencies.
- **Privacy-Centric:** No unnecessary tracking or heavy frameworks.

## 🏗️ Tech Stack
- **Languages:** Pure HTML5, CSS3, and Vanilla JavaScript.
- **Logic:** ES Modules (ESM) integration.
- **Frameworks:** None. (Zero dependency philosophy for maximum speed).
- **Environment:** Developed and managed via **Termux** on Android.
