# ⚡ ToolKit Pro

> **Free online tools for students and everyday use — No app download needed!**

🌐 **Live Site:** [toolkitpro.co.in](https://toolkitpro.co.in)

---

## 🛠️ Tools Included

| Tool | Description |
|------|-------------|
| 🧮 **Calculator** | Unlimited precision calculator — no scientific notation, supports very large numbers using Big.js |
| ♟️ **Chess** | Play chess vs AI (Easy / Medium / Hard) or vs a friend (2 Player mode) |
| 💱 **Currency Converter** | Real-time exchange rates for 160+ currencies worldwide |

---

## ✨ Features

- ⚡ No app download — works directly in the browser
- 👤 Guest mode — use all tools without signing up
- 🔐 Optional account system (Email + Google login via Supabase)
- 🌙 Dark mode & ☀️ Light mode
- 📱 Fully responsive — works on mobile, tablet, and desktop
- 📊 Google Analytics integrated
- 💰 Google AdSense ready

## 📁 File Structure

```
ToolkitPro-live/
├── index.html        # Main app (Calculator + Chess + Currency)
├── privacy.html      # Privacy Policy (standalone page)
├── terms.html        # Terms of Service (standalone page)
├── contact.html      # Contact Us (standalone page)
├── 404.html          # Custom 404 error page
├── sitemap.xml       # XML sitemap for SEO
├── robots.txt        # Search engine crawl rules
├── ads.txt           # Google AdSense verification
└── CNAME             # Custom domain config
```

## 🚀 Built With

- **HTML5** — Structure
- **CSS3** — Styling (no framework)
- **Vanilla JavaScript** — Logic (no framework)
- **[Big.js](https://github.com/MikeMcl/big.js/)** — Unlimited precision math
- **[Supabase](https://supabase.com)** — Auth & user management
- **GitHub Pages** — Hosting
- **Custom Domain** — `toolkitpro.co.in`
